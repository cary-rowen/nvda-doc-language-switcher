# Privacy Policy

NVDA Documentation Language Switcher does not collect, store, transmit, sell, or
share user data.

The extension runs only on supported NVDA documentation pages. It reads the
current page URL to build a corresponding English or local-language
documentation link, then inserts that link into the page.

The extension uses local page storage only to remember the most recently visited
non-English NVDA documentation language on the same site. This value stays in
the browser and is not transmitted anywhere.

No analytics, remote code, network requests, cookies, tracking identifiers, or
external services are used by the extension code.
