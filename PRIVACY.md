# Privacy

NVDA Documentation Language Switcher does not collect, store, transmit, sell, or
share user data.

The Chrome extension and Tampermonkey userscript run only on supported NVDA
documentation pages. They read the current page URL to build a corresponding
Chinese or English documentation link, then insert that link into the page.

No analytics, remote code, network requests, cookies, tracking identifiers, or
external services are used by the extension code.
